# Nome do Projeto: Sistema de Leilões

## Sobre o Projeto
O sistema de leilões foi desenvolvido para facilitar o cadastro, a pesquisa e a venda de itens que entram e saem mensalmente, prevenindo a perda de itens importantes.

## Tecnologias Utilizadas
- **Java:** Utilizado para todas as funcionalidades do sistema, exceto o banco de dados.
- **MySQL:** Utilizado para o banco de dados do sistema.
